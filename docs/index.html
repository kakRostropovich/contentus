Contentus.js

Небольшой плагин для легкой подстановки разного контента в интерфейс во время верстки.

Как работает плагин?

Этому элементу при верстке был задан класс "ct", и потому он сразу был выделен. Можно выделить любой другой элемент для подставновки контента зажав Alt и выбрав элемент курсором.

Изменять содержимое элемента можно клавишами "Вперед/Назад" или цифрами 1, 2, и 3 на клавиатуре.

Чтобы сбросить элемент в начальное состояние - можно нажать Escape
